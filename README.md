# patchwork_ruby

[![Build Status](https://travis-ci.org/joemccann/dillinger.svg?branch=master)](https://travis-ci.org/joemccann/dillinger)

This is the ROR solution of Patchwork tech task.

### Technology stack

* [Ruby 3.0.2]
* [Rails 6.1.4]
* [RSpec] - Rspec tests
* [Rubocop] - Ruby static code analyzer

### Installation

```sh
$ cd patchwork_ruby
$ bundle install
```

### Tests

```sh
$ cd patchwork_ruby
$ rspec spec
```
